# 五子棋对战 使用说明

## 简介

五子棋对战是一款经典的五子棋游戏，支持：
- **PvP 模式**：两人轮流在同一台电脑上对战
- **PvE 模式**：与电脑 AI 对战
- **LAN 模式**：局域网内两人对战（需要两台电脑）

游戏支持 2D 和 3D 两种视图模式。

---

## 安装与运行

### 方式一：直接运行（推荐）

1. 双击 `五子棋对战-便携版.exe` 文件
2. 等待几秒钟，游戏窗口会自动打开
3. 开始游戏！

> 无需安装任何软件，双击即可运行。

### 方式二：从源码运行（开发者）

需要先安装 Node.js，然后在命令行执行：

```bash
npm install
npm run dev
```

浏览器访问 http://localhost:3000

## 游戏操作

### 基本操作

| 操作 | 说明 |
|------|------|
| 点击棋盘 | 在交叉点落子 |
| 重置游戏 | 清空棋盘，重新开始 |
| 切换模式 | 选择 PvP / PvE / LAN |
| 切换视图 | 点击 "2D" 或 "3D" 按钮 |

### 游戏模式说明

#### PvP（本地双人）
- 黑方先手
- 两人轮流点击落子
- 先连成五子者获胜

#### PvE（人机对战）
- 玩家执黑先手
- 电脑执白
- 电脑会自动落子

#### LAN（局域网对战）
- 一人点击「创建房间」
- 将显示的房间号告诉对方
- 对方输入房间号点击「加入」
- 双方都准备好后游戏开始

### 3D 视图操作

| 操作 | 说明 |
|------|------|
| 鼠标左键拖拽 | 旋转棋盘视角 |
| 鼠标滚轮 | 缩放棋盘 |
| 点击棋盘 | 落子 |

## 局域网对战设置

### 查看本机 IP 地址

**Windows**：
1. 按 `Win + R`，输入 `cmd`，回车
2. 输入 `ipconfig`，回车
3. 找到「IPv4 地址」，类似 `192.168.x.x`

**Mac**：
1. 打开「系统设置」→「网络」
2. 查看当前连接的 IP 地址

### 让对方加入

1. 确保两台电脑连接同一个 WiFi 或路由器
2. 创建房间的玩家把 IP 地址和房间号告诉对方
3. 对方在浏览器输入 `http://你的IP:3000`
4. 对方输入房间号加入

## 常见问题

### Q: 双击 exe 没反应？
A:
- 可能被杀毒软件拦截，检查并允许运行
- 首次启动需要几秒钟加载，请耐心等待

### Q: 局域网对方无法连接？
A:
- 确认两台电脑在同一个局域网
- 检查 Windows 防火墙是否放行 3000 端口
- 尝试关闭防火墙测试

### Q: 3D 模式显示异常？
A:
- 更新浏览器或显卡驱动
- 使用 2D 模式

### Q: 想停止服务器？
A:
- 关闭游戏窗口即可
